# GitHub Copilot CLI - Dicee DevOps Config

Copilot CLI is configured for **DevOps and error checking** workflows in Dicee.

> **MCP Reference**: `docs/PROJECT-MCP-CONFIG.md` - Full tool decision matrix

## Quick Reference

### Error Checking
```bash
./scripts/copilot-mcp-wrapper.sh quality-gate    # Full 7-check gate
./scripts/copilot-mcp-wrapper.sh check-errors    # Quick error scan
./scripts/copilot-mcp-wrapper.sh fix-errors      # Auto-fix lint issues
```

### Deployment
```bash
./scripts/copilot-mcp-wrapper.sh deploy-do       # Deploy Durable Objects
./scripts/copilot-mcp-wrapper.sh deploy-pages    # Deploy Pages
./scripts/copilot-mcp-wrapper.sh deploy-full     # Full deployment
./scripts/copilot-mcp-wrapper.sh tail-logs       # Stream logs
```

### Environment
```bash
./scripts/copilot-mcp-wrapper.sh check-env       # Check credentials
./scripts/copilot-mcp-wrapper.sh status          # Project status
```

## MCP Tools

| Tool | Purpose |
|------|---------|
| `cloudflare-observability` | Worker logs for debugging |
| `cloudflare-builds` | Build status |
| `cloudflare-bindings` | KV/R2/D1 management |
| `akg` | Architecture validation |

## Quality Gate (7 Checks)

1. TypeScript & Rust type checking
2. AKG architectural invariants
3. Biome lint
4. Test suite
5. Secrets scan
6. Build verification
7. AKG diagram staleness

## Setup

Ensure credentials are loaded:
```bash
dicee-cf       # Cloudflare credentials
dicee-env dev  # Environment secrets
```

## Troubleshooting

**Wrangler auth failed**:
```bash
dicee-cf && wrangler whoami
```

**AKG server not responding**:
```bash
pnpm akg:discover
```
